## 📋 版本概述

ARBIG量化交易系统 v1.0.0 是第一个正式发布版本，标志着系统已经完成核心功能开发和充分测试，可以用于实盘交易。

---

## ✨ 主要特性

### 1. **完整的微服务架构**
- ✅ **交易服务** (Trading Service) - 端口 8001
  - CTP接口集成
  - 实时行情订阅
  - 订单管理和执行
  - 持仓查询和管理
  
- ✅ **策略服务** (Strategy Service) - 端口 8002
  - 策略引擎
  - K线数据管理
  - 技术指标计算
  - 信号生成和发送
  
- ✅ **Web管理服务** (Web Admin Service) - 端口 8003
  - 实时监控仪表盘
  - 策略管理界面
  - 交易操作界面
  - 系统配置管理

### 2. **核心交易策略**
- ✅ **MA-RSI组合策略** (MaRsiComboStrategy)
  - 双均线交叉信号 (EMA5/EMA20)
  - RSI指标确认 (14周期)
  - 完善的风控机制
  - CSV指标记录
  - 经过充分测试和优化

### 3. **风险控制系统**
- ✅ 止损/止盈机制
- ✅ 最大持仓限制
- ✅ 信号间隔控制
- ✅ 实时风控检查
- ✅ 交易时间控制

### 4. **数据处理能力**
- ✅ 实时行情数据处理
- ✅ 1分钟K线自动生成
- ✅ 技术指标实时计算
- ✅ CSV数据导出
- ✅ 交易日志记录

### 5. **用户界面**
- ✅ 现代化Web管理界面
- ✅ 实时数据更新
- ✅ 策略一键启停
- ✅ 持仓实时监控
- ✅ 交易历史查询

## 📦 技术栈

### **后端**
- Python 3.x
- vnpy (量化交易框架)
- CTP (期货交易接口)
- FastAPI (微服务框架)
- SQLite (数据存储)

### **前端**
- HTML5 + CSS3
- JavaScript (原生)
- Chart.js (图表库)

### **部署**
- Linux 服务器
- systemd 服务管理

## 🚀 快速开始

### **1. 启动系统**
```bash
# 启动所有服务
./start_all.sh

# 或分别启动
python services/trading_service/main.py &
python services/strategy_service/main.py &
python services/web_admin_service/main.py &
```

### **2. 访问Web界面**
```
http://localhost:8003
```

### **3. 注册和启动策略**
```bash
# 使用策略管理器
python strategy_manager.py

# 选择 "1 - 注册策略"
# 选择 "2 - GoldMaRsi (MA-RSI组合策略)"
# 选择 "2 - 启动策略"
```

## 📝 配置说明

### **主配置文件**
- `config/config.py` - 系统主配置
- `strategy_manager.py` - 策略配置

### **策略配置**
- `config/strategies/*.json` - 各策略的详细配置

### **关键参数**
```python
# 主力合约
main_contract = "au2512"

# MA-RSI策略参数
ma_short = 5          # 短期均线
ma_long = 20          # 长期均线
rsi_period = 14       # RSI周期
rsi_oversold = 30     # RSI超卖阈值
rsi_overbought = 70   # RSI超买阈值
trade_volume = 1      # 交易手数
max_position = 3      # 最大持仓
```
